# 🚀 Bharat FIL (Frontend)

A simple and intuitive frontend interface that helps users visualize and understand their investment portfolio using clear insights and analysis 📊.

---

## 📌 What it does

* Upload demat statement (PDF) 📄
* Display portfolio breakdown 📈
* Show risk levels (High / Moderate / Balanced) ⚠️
* Present insights in simple Hinglish 🗣️

---

## 🎯 Target Users

* Beginner & retail investors 👨‍💼
* Users who find financial data confusing 📉
* Anyone looking for simple portfolio insights 💡

---

## ⚙️ Tech Stack

* HTML
* CSS
* JavaScript

---

## 🧠 Key Idea

This frontend focuses on presenting complex financial data in a simple, clean, and user-friendly way.

---

## 🏆 Hackathon Project

Frontend developed for a hackathon project aimed at simplifying financial understanding 🇮🇳.
